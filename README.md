# PhotoWall - Android TV 专属复古照片墙

**PhotoWall** 是一款专为 Android TV 电视盒子和老旧安卓设备打造的轻量级电子相册应用。它能够将您闲置的电视或机顶盒变身为精美的动态照片墙，支持从 NAS (SMB) 网络共享或本地存储读取照片，通过优雅的动画效果循环播放。

## ✨ 核心特性

*   **📺 电视大屏优化**：专为遥控器操作设计的 UI，支持沉浸式全屏播放，隐藏状态栏和导航栏。
*   **🔌 双模数据源**：
    *   **SMB 网络共享**：直接连接群晖、威联通等 NAS 设备，无需下载照片到本地。
    *   **本地存储**：支持读取机身存储或外接 U 盘/移动硬盘中的照片。
*   **🎞️ 影院级转场**：
    *   **Ken Burns (推荐)**：经典的平移缩放效果，让静态照片动起来。
    *   **淡入淡出**：柔和的交叉渐变，适合安静的氛围。
    *   **滑动切换**：现代感的左右滑动切换。
*   **🚀 极致兼容与性能**：
    *   **向下兼容**：完美支持 **Android 4.4 (KitKat)** 及以上系统，复活家中的老旧机顶盒。
    *   **内存优化**：智能位图回收与降采样算法，在 1GB 内存设备上也能流畅运行 4K 照片播放。
    *   **防误触**：双击返回退出机制，防止家中老人小孩误操作。
*   **🔄 智能刷新**：
    *   **增量刷新**：自动比对服务器文件，仅更新新增或删除的照片，无需重新扫描全库。
    *   **全量刷新**：彻底重建缓存，确保数据一致性。

## 🛠️ 安装与使用

### 1. 安装应用
下载最新的 `release` APK 文件安装到您的 Android TV 或电视盒子中。

### 2. 初始配置
首次启动应用，点击遥控器 **菜单键** 或 **上方向键** 呼出设置菜单：

1.  **数据来源**：选择 "SMB 网络共享" 或 "本地存储"。
2.  **连接配置** (仅 SMB 模式)：
    *   输入 NAS 的 IP 地址 (如 `192.168.1.100`)。
    *   输入用户名和密码 (建议使用只读权限的账号)。
3.  **浏览路径**：点击浏览按钮，选择包含照片的文件夹 (支持子目录)。
4.  **播放参数**：设置幻灯片切换间隔和喜欢的转场风格。
5.  点击 **"保存并播放"**，尽情享受您的回忆！

### 3. 快捷键操作
*   **菜单键 / 上方向键**：呼出/隐藏控制菜单。
*   **确认键 / OK键**：暂停/继续播放。
*   **左/右方向键**：手动切换上一张/下一张照片。
*   **返回键**：
    *   在菜单中：关闭菜单。
    *   在播放中：双击退出应用。

## ⚙️ 开发构建

如果您是开发者，想要自行编译或修改代码：

### 环境要求
*   Android Studio Flamingo 或更高版本
*   JDK 1.8+
*   Android SDK API 33 (compileSdk)

### 构建步骤
1.  克隆本项目到本地：
    ```bash
    git clone https://github.com/your-username/PhotoWall.git
    ```
2.  使用 Android Studio 打开项目根目录。
3.  等待 Gradle Sync 完成。
4.  连接 Android 设备或模拟器，运行 `Run 'app'`。

### 主要依赖
*   **JCIFS-NG** (`eu.agno3.jcifs:jcifs-ng:2.1.9`): 强大的 SMB 协议客户端库。
*   **AndroidX MultiDex**: 确保兼容 Android 4.4 (API 19) 及以下版本。
